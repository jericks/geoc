filter cql2xml
==============

**Name**:

geoc filter cql2xml

**Description**:

Convert a CQL statement to an OCG XML Filter

**Arguments**:

   * -c --cql: The CQL statement

   * --help : Print the help message

   * --web-help : Open help in a browser



**Example**::

    geoc filter cql2xml -c STATE_ABBR=WA
