raster worldfile
================

**Name**:

geoc raster worldfile

**Description**:

Create a Raster world file

**Arguments**:

   * -b --bounds: The bounds

   * -s --size: The size

   * -f --file: The world file

   * --help : Print the help message



**Example**::

    geoc raster worldfile -b 0,0,10,10 -s 5,5 -f test.pgw
