style sld2ysld
==============

**Name**:

geoc style sld2ysld

**Description**:

Convert SLD to YSLD

**Arguments**:

   * -i --input: The input file or url

   * -o --output: The output file

   * --help : Print the help message

   * --web-help : Open help in a browser



**Example**::

    geoc style sld2ysld -i countries.sld -o countries.yml
