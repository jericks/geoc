style ysld2sld
==============

**Name**:

geoc style ysld2sld

**Description**:

Convert YSLD to SLD

**Arguments**:

   * -i --input: The input file or url

   * -o --output: The output file

   * --help : Print the help message

   * --web-help : Open help in a browser



**Example**::

    geoc style ysld2sld -i countries.yml -o countries.sld
