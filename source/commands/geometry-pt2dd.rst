geometry pt2dd
==============

**Name**:

geoc geometry pt2dd

**Description**:

Format a Point in Decimal Degrees

**Arguments**:

   * -p --point: The Point

   * -t --type: The output type (dms, dms_char, ddm, ddm_char)

   * --help : Print the help message

   * --web-help : Open help in a browser



**Example**::

    geoc geometry pt2dd -p "POINT (-122.5256194 47.212022222)" -t dms
